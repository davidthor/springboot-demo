You've deployed your app locally. Now what?

Below are additional steps you can try out with your newly deployed component.

  • Check out Hot-reloading
    1. Go to src/main/resources/templates/index.html in your project folder and open it.
    2. Uncomment line 11 to show the h1 tag, and save the file.
    3. Refresh the web page to see the change! For more info, checkout our docs.

  • Import the provided postman collection springboot.postman_collection.json
  • Try out the MovieRatingController, and refresh this page to see them update from the database service!
    1. Add a movie rating POST /movierating { title: string, rating: number }
    2. Get a movie rating GET /movierating/{id}
    3. Get ALL movie ratings GET /movierating
    4. Update a movie rating PUT /movierating/{id} { title: string, rating: number }
    5. Delete a movie rating DELETE /movierating/{id}

  • Deploy your app to the cloud via Architect (Recommended)
    1. Login or register a new Architect account for free at architect.io
    2. Authenticate using the CLI by running the command architect login
    3. Deploy your app to the cloud using the command architect deploy
